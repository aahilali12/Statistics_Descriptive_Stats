# 📊 Descriptive Statistics in Python

A robust, production-ready suite for performing foundational data analysis. This project automates the transition from raw, unorganized data into meaningful narratives using the industry-standard Python stack.

---

## 🚀 Project Overview
In any data science workflow, descriptive statistics are the essential first step. This repository provides a clean implementation of core measures to help you understand your data's "personality" before moving to complex modeling.

### Key Features
*   **Central Tendency:** Automated calculation of Mean, Median, and Mode to identify typical values.
*   **Dispersion Metrics:** Analysis of Variance, Standard Deviation, and Interquartile Range (IQR) to quantify data spread.
*   **Distribution Shape:** Measurement of Skewness and Kurtosis to identify asymmetry and tail-risk.
*   **Visualizations:** Publication-ready Histograms, Frequency Polygons, and Pie Charts using Matplotlib.

---

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Libraries:** 
    *   `NumPy`: High-performance numerical computations.
    *   `Pandas`: Data manipulation and summary statistics.
    *   `SciPy`: Advanced statistical functions (Skew/Kurtosis).
    *   `Matplotlib`: Data visualization.
